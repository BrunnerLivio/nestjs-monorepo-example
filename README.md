# NestJS mono-repo starter

[Nest](https://github.com/nestjs/nest) framework TypeScript monorepo starter repository.

## Installation

```bash
yarn
```

## Running the cli

```bash

yarn run start
# Open up on http://localhost:3000

# Run help
yarn run start --help

```
